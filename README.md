本次可视化分析应用整合舆情情感图、时序趋势图、商业数据走势图、用户属性统计图、文本词云、三维气泡图六大模块，共9组核心可视化图表，围绕电影《给阿嬷的情书》2026年4月15日—5月28日全周期数据开展分析。
情感饼图+情感时序折线图用于呈现微博、豆瓣双平台用户情感倾向随时间的变化；票房、排片多组时序趋势图，划分点映期、零点场、公映阶段，展示单日票房、票房占比、排片场次、排片占比的联动走势；各类别帖子占比的饼图用于展示各部分占整体的比例，方便探讨核心话题占据了多大的市场份额或讨论热度；文本分类柱状图与分阶段词云图谱，拆解全网讨论主题与内容演变；三维气泡图以转发数为X轴、评论数为Y轴、点赞数（气泡大小）为Z轴，结合颜色区分用户类型，多维还原不同群体的传播行为。整套可视化体系可直观判别影片热度来源、传播路径与市场驱动逻辑。
